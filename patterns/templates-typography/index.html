PatternFly Typography

Patternfly uses Overpass, an open-licensed alternative to Interstate that is specifically for use on-screen and in our software. Like Interstate, Overpass is based on the FHWA (Federal Highway Administration) series of fonts, which has been used for US Highway signage since 1945. Download Overpass Â»

Overpass can be used in a lot of the same ways as Interstate. We use Overpass for all web text, editable presentations, and product interfaces.

Vertical Rhythm

At the component level all vertical rhythm is removed. In cases where you have a section where you want to implement vertical rhythm apply the .pf-vertical-rhythm and all typeography level tags will receive a margin. All headings will also be given the appropriate size and weight. Be aware that mixing components and vertical rhythm could create unwanted margins and heading styles.
